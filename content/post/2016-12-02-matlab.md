## 同一张表作多图
``` matlab
%% Load data
data = csvread('td3.csv');

%% plot
fig = figure('position', [0, 50, 400, 600]);
subplot(311);
plot(data(:, 1), data(:, 2));
grid on;
title('373K G - X');

subplot(312);
plot(data(:, 1), data(:, 4));
grid on;
title('373K \Delta G - X');

subplot(313);
plot(data(:, 1), data(:, 5));
grid on;
title('373K \Delta^2 G - X^2');

[x1, x2] = findHIntercept(data(:, 1), data(:, 4))
%%% gtext(['Between ' num2str(x1) ' and ' num2str(x2)]);
%%% text(0.5, -4e5, 'test');

saveas(gcf, '373K.png');

```
